
Задача: Управление Галактическим Флотом
Описание задачи:

Разработайте систему управления галактическим флотом,
включающую следующие компоненты и функции:

Структуры и интерфейсы:

Создайте структуру Spaceship, которая будет
содержать информацию о космическом корабле 
(ID, название, модель, стоимость).

Создайте структуру Mission, которая будет 
содержать информацию о миссии 
(ID, название миссии, цель, дата начала).

Создайте интерфейс IFleetOperations, 
который будет содержать методы для управления
флотом (добавление корабля, удаление корабля, 
назначение миссии).

Классы и наследование:

Создайте класс Fleet, который будет 
реализовывать интерфейс IFleetOperations.

Создайте класс CommandFleet, который будет
наследовать класс Fleet и содержать информацию 
о командующем (имя, звание, опыт).

Коллекции и LINQ:

Используйте коллекции для хранения кораблей и миссий.
Реализуйте метод для поиска кораблей по заданному
критерию (например, по модели или диапазону стоимости)
с использованием LINQ.

Реализуйте метод для получения всех миссий 
определенного корабля.

Работа с JSON:

Реализуйте методы для сериализации и 
десериализации кораблей и миссий в/из JSON.
